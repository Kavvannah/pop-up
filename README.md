# pop-up

A simple application that uses 'event listeners' and 'query selectors' to add and/or remove a popup display. This project was made using HTML, CSS, and Javascript.

## Project Status
This project is complete. Users can click the displayed button and a popup will appear with functionality that can be exited.

## Installation and Setup Instructions

#### Example:  

Clone down this repository. You will need `node` and `npm` installed globally on your machine.  

Installation:

`npm install`  

To Run Test Suite:  

`npm test`

`npm start`
